# ConversaCore AI 🤖🧠  
A multi-model AI chatbot platform designed for advanced conversational experimentation. ConversaCore AI enables dynamic model selection, persona-based interactions, and real-time responses, creating a flexible environment to explore how different AI systems think, respond, and adapt.

---

## 🚀 Features  
- 🤖 **Multi-Model Support** — Switch between different AI models dynamically  
- 🎭 **Persona-Based Responses** — Customize tone and behavior of the AI  
- 🧠 **Chat Memory** — Maintains context for more natural conversations  
- 🎨 **Theme Switching** — Light/Dark UI modes for better experience  
- 💾 **Export Tools** — Save conversations as text or markdown  
- ⚡ **Real-Time Interaction** — Fast, responsive messaging system  
- 🔁 **Intelligent Fallback** — Automatically switches models on failure  

---

## 🧠 Tech Stack  
- 🐍 **Python** — Core logic and backend  
- 🪟 **Tkinter** — GUI framework  
- 🤖 **Gemini / Multi-Model APIs** — AI response generation  
- ⚙️ **Async Processing** — Smooth real-time performance  
- 💾 **Local Storage** — Chat history and session handling  

---

## ⚙️ Setup  

1. Open the project files  
2. Locate where the API key is defined  
3. Replace:
   ```
   GEMINI_API_KEY
   ```
   with your own API key:
   ```
   YOUR_API_KEY
   ```

4. Run the application:
   ```bash
   python main.py
   ```

---

## 💡 How It Works

1. User selects model and persona
2. Input is processed and sent to selected AI
3. System maintains conversation context
4. Response is generated and displayed instantly
5. Fallback system activates if primary model fails
